CM_RT_API int32_t CreateCmDevice ( MOS_CONTEXT *  mosContext,
CmDevice *&  device,
uint32_t  devCreateOption,
uint8_t  priority 
)

Creates a CmDevice from a MOS context.

If an existing CmDevice has already associated to the MOS context, the existing CmDevice will be returned. Otherwise, a new CmDevice instance will be created and associatied with that MOS context.

Parameters
mosContext[in] pointer to MOS conetext.
device[in,out] reference to the pointer to the CmDevice.
devCreateOption[in] option to customize CmDevice.
Return values
CM_SUCCESSif the CmDevice is successfully created.
CM_NULL_POINTERif mosContext is null.
CM_FAILUREotherwise.
