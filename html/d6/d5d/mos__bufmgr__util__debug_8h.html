Detailed Description

Bufmgr Debug and Print utilities.

Provides assert and print to debug console functionality All Bufmgr debug and print utilities will only function in debug or release internal drivers, in a release driver they will be NOPs. To enable log in mos bufmgr by bellow config: [config] Mos Message Tags=4 Mos Sub Components Tags=15728640 //uint32_t(24:21) -> (F00000) Mos Tags By Sub Component=1
